# 🐍 Snake Game

A classic Snake Game implemented using JavaScript, HTML, and CSS. Control the snake to eat food, grow in length, and avoid colliding with walls or itself.

## 🎮 Play the Game

Experience the game live at: [shai360hai.github.io/snake/](https://shai360hai.github.io/snake/)

## 🚀 Features

- **Responsive Design**: Enjoy the game on various devices with adaptive layouts.
- **Smooth Controls**: Navigate the snake seamlessly using arrow keys.
- **Score Tracking**: Keep track of your progress with an intuitive scoring system.

## 🛠️ Technologies Used

- **JavaScript**: Core game logic and interactivity.
- **HTML5**: Structuring the game interface.
- **CSS3**: Styling and responsive design.

## 📦 Installation

To run the game locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shai360hai/snake.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd snake
   ```
3. **Open `index.html`** in your preferred web browser.

## 🎯 How to Play

- **Start the Game**: Open the game link or `index.html` file.
- **Control the Snake**: Use the arrow keys to move:
  - Up: ↑
  - Down: ↓
  - Left: ←
  - Right: →
- **Objective**: Guide the snake to eat the food that appears randomly on the grid. Each piece of food eaten increases the snake's length by one unit and adds 10 points to your score.
- **Game Over**: The game ends if the snake collides with the walls or itself.
